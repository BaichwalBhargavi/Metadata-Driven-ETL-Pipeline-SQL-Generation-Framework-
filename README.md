# Metadata-Driven-ETL-Pipeline-SQL-Generation-Framework-

This project is a metadata-driven ETL framework built using PySpark, designed to create scalable, reusable, and configurable data pipelines without hardcoded business logic. The framework leverages an object-oriented SQL Builder and a centralized SQLite metadata repository to dynamically generate queries, process data, and export results in multiple formats.

The solution demonstrates modern Data Engineering practices such as metadata-driven development, dynamic query generation, schema abstraction, and configurable ETL orchestration.
